# Machine Learning Practice Notebooks

This repository contains a collection of Jupyter notebooks for practicing various machine learning algorithms and techniques. Each notebook demonstrates a specific model or method using real-world datasets.

## Contents

- **Classification**
  - Decision Trees: `ML0101EN-Clas-Decision-Trees-drug.ipynb`
  - K-Nearest Neighbors: `ML0101EN-Clas-K-Nearest-neighbors-CustCat.ipynb`
  - Logistic Regression: `ML0101EN-Clas-Logistic-Reg-churn.ipynb`
  - Support Vector Machines: `ML0101EN-Clas-SVM-cancer.ipynb`

- **Clustering**
  - DBSCAN: `ML0101EN-Clus-DBSCN-weather.ipynb`
  - Hierarchical Clustering: `ML0101EN-Clus-Hierarchical-Cars.ipynb`
  - K-Means: `ML0101EN-Clus-K-Means-Customer-Seg.ipynb`

- **Regression**
  - Simple Linear Regression: `ML0101EN-Reg-Simple-Linear-Regression-Co2.ipynb`
  - Multiple Linear Regression: `ML0101EN-Reg-Mulitple-Linear-Regression-Co2.ipynb`
  - Polynomial Regression: `ML0101EN-Reg-Polynomial-Regression-Co2.ipynb`
  - Nonlinear Regression: `ML0101EN-Reg-NoneLinearRegression.ipynb`

- **Recommender Systems**
  - Collaborative Filtering: `ML0101EN-RecSys-Collaborative-Filtering-movies.ipynb`
  - Content-Based Filtering: `ML0101EN-RecSys-Content-Based-movies.ipynb`

## Usage

1. Open any notebook in Jupyter or VS Code.
2. Follow the instructions and run the cells to explore the models.
3. Modify code and experiment with different parameters and datasets.

## Requirements

- Python 3.x
- Jupyter Notebook or VS Code
- Common ML libraries: `numpy`, `pandas`, `scikit-learn`, `matplotlib`, etc.
